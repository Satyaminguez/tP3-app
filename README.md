# TP3 — Application Data Interactive

## Description
Cette application permet d’explorer des données Open Data issues d’un pipeline
de géocodage (TP2), via une interface Streamlit interactive et un chatbot IA.

## Fonctionnalités
- Chargement de données Parquet
- Visualisations interactives (Plotly)
- Indicateurs clés
- Chatbot IA pour interroger les données

## Lancer l’application

```bash
ollama run mistral
uv run streamlit run app_streamlit.py
```
